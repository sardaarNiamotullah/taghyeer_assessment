## Getting Started (Local Setup)

**Requirements:** Flutter SDK `^3.10.7`, Dart SDK `^3.0`

```bash
git clone https://github.com/sardaarNiamotullah/taghyeer_assessment.git
cd taghyeer_assessment
flutter pub get
flutter run
```

Dependencies used: `flutter_bloc: ^9.1.1`, `http: ^1.6.0`, `shared_preferences: ^2.5.4`
